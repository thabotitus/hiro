TTS | WEB STARTER KIT

TTS Web Starter Kit is, as the name suggests, a starter kit to get you up and running with a static site generated from from Nunjucks templates and Sass. This can be use useful when you you want to build UI kits, websites or static component libraries.

Free Download Github Repo

Features

It's a small package that delivers great value

All features come out the box without any additional configuration.

Hot Reloading

Make changes to your (s)css files without the need do a page reload. This creates a significant time-savings when building complex designs.

Live Reloading

Page automatically refreshes after you make changes to your html, eliminating the need to constantly switch tabs to refresh your page, again saving you time and effort.

Image Optimization

Optimizes all your images on the fly, even while you continue to make changes. Useful for faster loading pages that have lots of images.

Package & Zip

Your code ready for distribution? Zip it and ship it with a simple command.

Version Management

Easily bump your code versions with a simple command.

Easy Deploy

Deploy your site to Surge.sh in seconds, for free with a simple command. A great way to get feedback on your work just by sharing a link.

GH Pages ready

Push your code to Github and it will be ready to be served by Github Pages.

Stimulus JS ready

We don't tell you how to write your javascript, but we have included a simple way for you to add javascript with StimulusJs.
